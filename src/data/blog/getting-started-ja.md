# タロット初心者ガイド

タロットウェブへようこそ！タロットが初めての方も、復習したい方も、このガイドで必要なことをすべて説明します。

## タロットとは？

タロットは78枚のカードからなるデッキで、それぞれに象徴的なイメージと意味が込められています。デッキは主に2つのグループに分かれています：

- **大アルカナ**（22枚）：人生の重要な出来事、精神的な教訓、普遍的な力を表します。*愚者*、*恋人*、*死神*のようなカードは、人間の経験における大きなテーマを扱います。
- **小アルカナ**（56枚）：日常的な状況をカバーし、4つのスートに分かれています——カップ（感情）、ワンド（行動）、ソード（知性）、ペンタクル（物質世界）。

## このサイトの使い方

1. **質問を考える**——恋愛、キャリア、自己成長、あるいは単に「今日は何に集中すべきか？」でも構いません。空白のままにして、カードに一般的なガイダンスを任せることもできます。
2. **スプレッドを選ぶ**——3つのオプションがあります：
   - *シングルカード*：今この瞬間のクイックな洞察。
   - *過去・現在・未来*：過去の出来事が現在をどのように形作っているか、そしてこの先何が待っているかを見ます。
   - *問題・障害・助言*：核心的な問題を特定し、前進する道を見つけます。
3. **カードを引く**——ボタンをクリックして、シャッフルアニメーションを見てください。カードが一枚ずつ表示されます。
4. **解釈を読む**——各カードには、スプレッド内の位置に応じた意味が付いています。

## いくつかのヒント

- タロットを使うのに神秘的なものを信じる必要はありません。多くの人が自己反省のツールとして使っています——カードは状況を別の角度から考える助けになります。
- 「悪い」カードはありません。*死神*や*塔*のようなカードでさえ、文字通りの破滅ではなく、変容と必要な変化を表しています。
- 逆位置のカードは、しばしばブロックされたエネルギー、内面的な課題、またはカードの意味の別の側面を指します。
