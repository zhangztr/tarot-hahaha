# 塔罗新手入门指南

欢迎来到塔罗秘境！无论你是完全的新手还是想复习基础知识，这篇指南将带你了解所需的一切。

## 什么是塔罗牌？

塔罗牌共有 78 张，每张牌都有其独特的象征意义和画面。整套牌分为两大类：

- **大阿尔卡纳**（22 张）：代表重大人生事件、精神课题和原型力量。像*愚人*、*恋人*、*死神*这样的牌涉及人生中的大主题。
- **小阿尔卡纳**（56 张）：涵盖日常生活中的各种情况，分为四个牌组——圣杯（情感）、权杖（行动）、宝剑（理智）和钱币（物质）。

## 如何使用本站

1. **想一个问题**——可以是关于爱情、事业、个人成长，或者简单地"今天应该关注什么？"也可以什么都不填，让牌给出一般性的指引。
2. **选择一个牌阵**——我们提供三种选择：
   - *单张牌*：当下处境的快速指引。
   - *过去·现在·未来*：了解过去如何影响现在，以及未来可能的走向。
   - *问题·阻碍·建议*：识别核心问题，找到前进方向。
3. **抽牌**——点击按钮，观看洗牌动画。你的牌会一张一张亮出来。
4. **阅读解读**——每张牌在牌阵的特定位置上都有对应的含义解读。

## 一些小建议

- 你不需要相信任何神秘主义的东西也可以使用塔罗。很多人把它当作一种自我反思的工具——牌能帮助你从不同角度思考自己的处境。
- 没有"坏牌"。即使是*死神*或*高塔*这样的牌也代表转变和必要的改变，而不是字面意义上的厄运。
- 逆位的牌（倒过来的牌）通常指向受阻的能量、内心的挑战，或牌义的另一种面向。
